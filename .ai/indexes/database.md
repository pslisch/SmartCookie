# Database Index

This index describes the data models, entity relationships, and schemas supporting the backend databases.

---

## 🗄️ Database Schema Specification

Every database table or collection should document:
- **Tables / Collections**: Table name and field models.
- **Relations**: Foreign keys and links.
- **Migrations**: Incremental migration log details.
- **Used By**: API endpoints or server workers calling the table.
